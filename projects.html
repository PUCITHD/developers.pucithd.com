pucithd.com

GitHub: https://github.com/PUCITHD/pucithd.com

General

pucithd.com is main website of PUCIT Help Desk which host tools and mini projects on it, details of whom are given below. Architecture used in developing these tools is MVC. .json file as Model, .html as View and .js as Controller.

Jobs

jobs.json in data directory acts as Model. If you want to post a job on PUCIT Help Desk's Job Board simply edit json file to add job to job board.

jobs.html in main directory acts as View. All jobs from json file are render in this view.

jobs.js in js directory acts as Controller. This file put jobs on job board by fetching them from json file.

Tools

Tools can be developed for ease of use for students, applicants and alumni. Currently there are two functional tools, Aggregate Calculator and GPA Calculator while Preference Setter has been depreciated.

Others

You can freely contribute to PUCIT Help Desk's any project by creating/enhancing any module.

PUCITx

Github: https://github.com/PUCITHD/PUCITx

PUCITx is an open source learning platform to learn from faculty of PUCIT. You can add course on PUCITx by adding a package folder in courses_data folder.

Package details are as follow.


|-COURSE_TITLE_SLUG
|--outline.json
|--social_photo.png

COURSE_TITLE_SLUG is used for urls of course e.g. for course of computer vision COURSE_TITLE_SLUG will be computer-vision.

social_photo.png is a thumbnail photo to appear on website and social media platforms.

outline.json core file in any course which is being rendered by mooc. it has course title, youtube video ids, time duration of each video. For sample, Click Here.

Blogs

GitHub: https://github.com/PUCITHD/blog.pucithd.com

blog.pucithd.com is a platform having blogs by PUCIT Alumni and Students.

Frontend: Frontend of blog.pucithd.com is built with Angular 6 and is powered by Github Pages. Angular 6 consume APIs of wordpress.com.

Backend: Backend is powered by wordpress.com.

Developers Portal

GitHub: https://github.com/PUCITHD/developers.pucithd.com

developers.pucithd.com is developer portal has guidelines to contribute to PUCIT Help Desk through code. Currently no framework or standard is being followed in maintaining developers.pucithd.com
