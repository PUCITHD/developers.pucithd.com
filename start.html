Quick Start

Introduction

To contribte to PUCIT Help Desk you need to know working knowledge of git and a GitHub account. Sir Bilal Shahzad have made a beautiful tutorial on use of git. We will recomend you to go through that once.

Take Course

Installation

Linux

Install Git on your local machine.

For Ubuntu/Debian:

sudo apt update

sudo apt install git

For CentOS:

sudo yum install git

Windows/MacOS

For Windows and MacOS GitHub have a desktop client. You can download that from here.

Next Steps

Now you have environment ready to start contributing to PUCIT Help Desk. You can see the list of projects to contribute here.
